# Descripción del Proyecto

Este proyecto implementa un sistema académico para la gestión de estudiantes utilizando la estructura de datos **Árbol Binario de Búsqueda (BST - Binary Search Tree)**.

El sistema permite almacenar, organizar y administrar estudiantes utilizando como clave principal su número de cédula, permitiendo realizar operaciones de inserción, búsqueda, eliminación y distintos recorridos del árbol.

Cada estudiante contiene la siguiente información:

- Cédula
- Apellidos
- Nombres
- Nota final
- Carrera
- Nivel

---

# Objetivo General

Implementar un sistema de gestión académica utilizando árboles binarios de búsqueda en C++, aplicando estructuras dinámicas, recursividad y algoritmos de recorrido.

---

# Objetivos Específicos

- Implementar operaciones básicas sobre árboles binarios de búsqueda.
- Aplicar recorridos DFS y BFS para visualizar la información almacenada.
- Analizar la eficiencia de los árboles binarios en la organización de datos académicos.

---

# Funcionalidades Implementadas

## Gestión de estudiantes

- Insertar estudiante
- Buscar estudiante por cédula
- Eliminar estudiante

## Recorridos del árbol

- Recorrido Inorden
- Recorrido Preorden
- Recorrido Postorden
- Recorrido por niveles (BFS)

## Operaciones complementarias

- Contar estudiantes registrados
- Calcular altura del árbol
- Buscar estudiante con mayor nota
- Buscar estudiante con menor nota
- Mostrar estudiantes aprobados
- Mostrar estudiantes reprobados
